# Cartographie avec R Leaflet

Les bases de la cartographie avec le package R "leaflet".

*Leaflet est une bibliothèque JavaScript libre de cartographie en ligne, elle est notamment utilisée par le projet de cartographie libre et ouverte OpenStreetMap, ainsi que par exemple par "The New York Time", "The Washington Post", GitHub ou encore Flickr.
R dispose d'un package simplement intitulé "leaflet" qui permet de réaliser des cartes dynamiques tout en codant en R mais avec la puissance et la diversité offerte par le package JS original.*

Voir l'article de blog correspondant : http://www.dacta.fr/blog/cartographie-r-leaflet.html
